About

This project collects open source datasets focused on Italy and uses various JavaScript libraries to visualize the data in easily understandable graphs. The datasets are first processed using python3 pandas library. The python scripts written export json files that contain only the information needed to display the data. The majority of the graphs are built as SVG elements with the JavaScript D3 library.

The motivation for this project came from an interest in data visualization using D3.js. Additionally, this project was also used to learn Tailwind CSS. The inspiration to use datasets about Italy is simply from interest in the country.
